# Quick Start - 5 Minutos

## 1. Instale FASE

```bash
npx fase-ai
```

Escolha: Claude Code, Local (projeto atual).

## 2. Inicie um Novo Projeto

```bash
/fase-novo-projeto
```

FASE vai fazer perguntas para entender seu projeto:
- O que você quer construir?
- Qual é a visão geral?
- Quais tecnologias usar?

## 3. Veja o Roadmap

```bash
/fase-progresso
```

Isso mostra o roadmap de fases para seu projeto.

## 4. Planeje a Primeira Fase

```bash
/fase-planejar-fase
```

FASE cria um plano detalhado (PLAN.md) com:
- Tarefas específicas
- Dependências
- Critérios de sucesso

## 5. Execute a Fase

```bash
/fase-executar-fase
```

FASE implementa tudo automaticamente com:
- Commits atômicos
- Checkpoints persistentes
- Tratamento de desvios

## 6. Verifique o Resultado

```bash
/fase-verificar-trabalho
```

FASE valida que a fase atingiu seus objetivos.

## Próximos Passos

- Explore [Todos os Comandos](/FASE/docs/reference/commands/)

## Fluxo Típico

```
novo-projeto
  → progresso
  → planejar-fase
  → executar-fase
  → verificar-trabalho
  → (repetir para próximas fases)
  → completar-marco
```

**Pronto!** Você agora está usando FASE para automação real.
